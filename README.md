# OlivmonManagement
I developed this android app as management system of <a target="_blank" href="https://github.com/farukcuha/OlivmonApp">OlivmonApp</a>. The manager can control <a target="_blank" href="https://github.com/farukcuha/OlivmonApp">OlivmonApp</a>. This app provide order control system, product add-delete system, main screeen item changing control system and users seeing system.
